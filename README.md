# ESP32 DHT11 & TCS34725 Sensor Project

This project demonstrates interfacing an ESP32 with:
- DHT11 temperature & humidity sensor
- TCS34725 RGB color sensor
- Built-in LED blinking using millis()

## Hardware Used
- ESP32 Dev Board
- DHT11 Sensor
- TCS34725 Color Sensor

## Libraries Required
- DHT sensor library
- Adafruit Unified Sensor
- Adafruit TCS34725
- Wire (default)

## Features
- Non-blocking LED blink
- Temperature & humidity monitoring
- RGB, Lux, and Color Temperature reading

## Serial Output
Displays sensor values every 2 seconds at 115200 baud.
